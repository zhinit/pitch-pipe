# Concert Pitch (A440)

A440 is the standard tuning reference: the note A4 is defined as 440 Hz. It is
formalized as ISO 16. (source: wikipedia-a440-pitch-standard.md)

## Standard

- A4 = 440 Hz
- ISO 16 (adopted 1955, reaffirmed 1975)
- Internationally adopted at a 1939 conference in London

(source: wikipedia-a440-pitch-standard.md)

## History

- 1834: Scheibler proposed A440 in Stuttgart
- 1926: American music industry informally adopted 440 Hz
- 1936: American Standards Association formally recommended A440
- 1939: International conference officially adopted A440
- 1955: ISO adopted as Recommendation R 16
- 1975: Reaffirmed as ISO 16

(source: wikipedia-a440-pitch-standard.md)

## Variations in Practice

| Context                    | Frequency |
|----------------------------|-----------|
| ISO 16 standard            | 440 Hz    |
| Continental European orch. | 440–444 Hz |
| Modern baroque pitch       | 415 Hz    |
| Chorton (German church)    | 466 Hz    |
| Classical pitch            | 427–430 Hz |

(source: wikipedia-a440-pitch-standard.md)

## Relevance to This Project

A pitch pipe gives reference tones for tuning. The standard is A440. The app
should use 440 Hz for A4, with all other notes derived via
[[equal-temperament]] (12-TET).
