# Connect IQ Store Asset Requirements

Source: https://developer.garmin.com/brand-guidelines/connect-iq/
Fetched: 2026-06-17

## App Store Icon (Primary)

- Dimensions: 500×500 px
- Color space: sRGB
- Allow 10px padding from edge
- Keep designs simple — complex details are hard to see at small sizes
- No black or transparent backgrounds
- No descriptive text, clip art, or fine details
- No Garmin branding without permission
- Use a simple, solid background color

## On-Device Icon

- Dimensions: 128×128 px
- Color space: sRGB
- Two versions supported:
  - Low-color: 64-color palette (for MIP displays like Fenix)
  - High-color: 24-bit per pixel (for AMOLED displays like Epix, Venu)

## Hero Image

- Dimensions: 1440×720 px
- Upload separate versions for different languages if text is included

## Launcher Icon (In-App)

- Size varies by device (36×36 to 65×65)
- Specified per-device in the SDK
- If the app icon doesn't match the device size, it will be auto-scaled

## Screenshots

- Take screenshots from the simulator for each device category
- Should show the app in action
- "Be very specific in your description"

## Activity Type

- Must categorize the app by functionality
- Select all applicable activity categories
