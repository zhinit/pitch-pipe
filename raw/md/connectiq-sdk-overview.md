# Connect IQ SDK

Source: https://developer.garmin.com/connect-iq/sdk/
Fetched: 2026-06-14

## Current Version

Connect IQ 9.2.0 SDK (Last Updated: June 08, 2026)

This version supports Connect IQ 1.x, 2.x, and 3.x.

## Download Links

The SDK Manager is available for three platforms:

- **Windows**: SDK Manager zip file
- **Mac**: SDK Manager DMG installer
- **Linux**: SDK Manager zip file

All downloads require acceptance of the Connect IQ Developer Agreement.

## Installation Process

Install the SDK Manager first, then use it to download the Connect IQ SDK and
compatible device simulators. Launch the manager, complete initial
configuration, and select the downloaded SDK as the active version.

## Development Environment

The Visual Studio Code Monkey C extension is available through the VS Code
marketplace. A direct download option exists for developers preferring
alternative IDEs.

## Mobile SDKs

Companion SDKs are provided separately:

- **Android**: Distributed via Maven Central with GitHub sample application
- **iOS**: Available through GitHub with example implementation code

## What's Included

Sample code, simulators, tools, and libraries, along with documentation and
APIs for building Connect IQ applications using the Monkey C programming
language.
