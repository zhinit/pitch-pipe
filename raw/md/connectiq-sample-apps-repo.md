# Garmin Connect IQ Apps Repository

Source: https://github.com/garmin/connectiq-apps
Fetched: 2026-06-14

## Directory Organization

The repository contains six main folders:

1. **audio-provider** — Special apps for device media player audio content
2. **barrels** — Monkey Barrels (shareable code libraries)
3. **datafields** — Apps running within native activities
4. **device-apps** — Full-featured third-party activity apps
5. **watchfaces** — Low-power home screen replacements
6. **widgets** — At-a-glance accessible apps
